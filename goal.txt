SOLVER SETTINGS (all mandatory)
-------------------------------
1. ONE engine config for ALL biases. No U/L mixing. The 2026-07-19 paket
   run mixed two configs (see mqw_index_merge_manifest.json) and the
   dAlpha(V) trend breaks exactly at the config seam (-1.3 to -2.1 V).
   Keep: 8x8 k.p, 1-QW partitions, dEc 96 meV anchor. Use whichever
   wf-location-filter setting solves every bias without NaN bands, but
   the SAME setting everywhere.
2. Bias list: exactly the 5 target biases: -0.5, -1.1, -1.7, -2.1, -2.5 V.
   -0.5 V is the reference and is currently missing from paket/per_bias.
3. Coupled CHARGE -> MQW solve (potential profile from CHARGE). Mandatory.
4. Temperature: TWO runs of the same 5-bias set.
   a) 323 K (50 C): this is the match run; the ER measurement is at 50 C.
   b) 303 K (30 C): comparison run, same config, same biases.
   Note: there is NO 30 C measurement in measurement_data (all ER files
   are 50 C), so the 30 C run is compared against the 50 C run, not
   against an experiment: it isolates the pure temperature effect on the
   edge position and on dAlpha(1290). Expected: at 30 C the gap widens,
   the edge sits blue of the 50 C edge, dAlpha(1290) drops.
   Verify which temperature the current paket run used; 300 K would
   shift the edge by itself and would explain part of any mismatch.
5. Wavelength grid: keep 1150-1400 nm at 1 nm spacing.

EVALUATION RECIPE (fixed in advance, do not adjust afterwards)
--------------------------------------------------------------
- alpha_TE at the ridge-centre column, averaged over the 16 MQW sample
  points, read at 1290.1 nm.
- dAlpha(V) = alpha(V) - alpha(-0.5 V).
- ER_pred(V) = -4.343 * 0.128 * 0.008 cm * dAlpha(V).

-------------------------------------
- dAlpha(V) strictly monotonic in |V| (the current run fails exactly this).
- dAlpha(-2.5 V) = 1793 cm^-1 within +/-15 percent.
- RMS over the 5 biases <= 150 cm^-1.
- Exciton peak reaches 1288-1292 nm near -2.5 V (the current run already
  does this; do not lose it).
The 30 C run has no pass/fail criterion; report edge position and
dAlpha(1290) per bias next to the 50 C values.
